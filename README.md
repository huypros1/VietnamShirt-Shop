## Tính năng chính

- **Dành cho Khách hàng:** Đăng ký/đăng nhập an toàn, duyệt và tìm kiếm sản phẩm, thêm vào giỏ hàng, thanh toán (checkout), quản lý hồ sơ, danh sách yêu thích và khôi phục mật khẩu qua Email (PHPMailer).
- **Dành cho Quản trị viên:** Trang Dashboard riêng biệt giúp quản lý toàn diện các danh mục, sản phẩm, trạng thái đơn hàng và tài khoản người dùng.
